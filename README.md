Задание 1: Базовые рекурсивные функции
Вариант 1. Напишите рекурсивную функцию для вычисления суммы цифр числа.


анализ алгоритма 

Суть задачи: для заданного целого числа вычислить сумму его цифр (например, для числа 123 сумма = 1 + 2 + 3 = 6).
Идея рекурсии:
  Базовый случай (условие остановки): если число < 10, то его единственная цифра — это и есть сумма.
  Рекурсивный шаг: сумма цифр числа n равна последней цифре (n % 10) плюс сумма цифр оставшейся части (n // 10).

  
  Пример на Python (пошагово):
# Базовый случай: если n < 10, возвращаем n
if n < 10:
    return n
# Рекурсивный вызов: последняя цифра + сумма цифр остатка
return (n % 10) + sum_digits(n // 10)
Трассировка для n = 123:
    sum_digits(123) → 3 + sum_digits(12)
    sum_digits(12) → 2 + sum_digits(1)
    sum_digits(1) → 1 (базовый случай)
    Итог: 3 + 2 + 1 = 6
    
Временная сложность
O(d), где d — количество цифр в числе n.
Почему:
    Каждый рекурсивный вызов обрабатывает одну цифру (отсекает последнюю via n // 10).
    Глубина рекурсии = количеству цифр d (например, для 123 → 3 уровня).
    На каждом уровне выполняется константное число операций (%, //, +).

Замечание:
    d = ⌊log₁₀n⌋ + 1 → сложность также можно записать как O(log n) по основанию 10.
