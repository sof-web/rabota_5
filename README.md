Чтобы рекурсивно вычислить сумму цифр числа n:
Если число равно 0 — сумма равна 0 (базовый случай).
Иначе:  
  Получить последнюю цифру: n % 10  
  Отбросить последнюю цифру: n // 10  
  Рекурсивно вызвать функцию для укороченного числа и прибавить последнюю цифру.

Для отрицательных чисел можно взять модуль (abs(n)), так как сумма цифр не зависит от знака.

Объяснение шагов с примерами кода на Python
1. Базовый случай (остановка рекурсии):  
   Когда число становится 0, рекурсия завершается.  

   if n == 0:
       return 0
   
2. Извлечение последней цифры:  
   Остаток от деления на 10 даёт последнюю цифру.  

   last_digit = n % 10
   

3. Рекурсивный вызов для остальных цифр:  
   Целочисленное деление на 10 отбрасывает последнюю цифру.  


   return last_digit + digit_sum(n // 10)

Временная сложность

- O(d), где d — количество цифр в числе n.
- Поскольку в каждом рекурсивном вызове число уменьшается в 10 раз (n → n // 10), количество вызовов = количеству цифр.
- Количество цифр d ≈ log₁₀(n) + 1, но принято выражать сложность как O(d) (линейно от длины числа в цифрах).

Это одинаково для всех входов — нет различия между лучшим/худшим случаем, потому что каждый раз обрабатываются все цифры.
